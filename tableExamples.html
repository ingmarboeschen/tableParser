Extraction of tabled statistical results with tableParser

Appendix A: Example tables



This HTML file may be processed with the tableParser R package. See: /cran/tableParser


- To convert the tables within this document into R character matrices run:

table2matrix(`tableExamples.html`)

- To convert the reported results in each table into a text string run:

table2text('tableExamples.html`)

- To extract the statistical standard results as data frame object with recalculated p-values run:

table2stats(`tableExamples.html`)

See the documentation of each function for further options.
